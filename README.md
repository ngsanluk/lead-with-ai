# What is Generative AI?

![](https://encrypted-tbn0.gstatic.com/images?q=tbn:ANd9GcRVfmlpwY_rU9vxiohYexQSSlfhKZasvs5p9Q&s)

- Generative AI is a type of artificial intelligence that can **create new content**, like text, images, or music, in response to prompts from users.
- It uses machine learning models trained on large datasets (**LLM** - **L**arge **L**anguage **M**odels) to generate outputs that mimic human creativity.

![](https://media.licdn.com/dms/image/v2/D5612AQGAsURTO7PQxw/article-cover_image-shrink_720_1280/B56Zc0Y8tNGoAM-/0/1748930653865?e=2147483647&v=beta&t=tdRXiHj_1cyBP2nr-LrzVUNRzCFfo_TBisx7sm4sx5A)

# Reminders on Using AI Tools

- AI is not always correct
- AI sometimes generated false information (known as **hallucination**)
- Free AI tools may collect and use your data to train their models
- Generative AI tools updates frequently, so features may change and user interfaces may look different
- Human in the loop

# Effective Prompting Structure: RICE FACT Framework

| Element                 |     | Description                                                                            |
| ----------------------- | --- | -------------------------------------------------------------------------------------- |
| **R**ole                | ‚Üí   | Define the role of the AI (e.g., "You are a marketing expert.")                        |
| **I**nstruction/Task    | ‚Üí   | Specify the goal of the interaction (e.g., "Help me create a marketing plan.")         |
| **C**ontext/Explanation | ‚Üí   | Provide background information (e.g., "Our target audience is young professionals.")   |
| **E**xamples            | ‚Üí   | Give examples of desired output (e.g., "Include social media strategies.")             |
| **F**ormat/Layout       | ‚Üí   | Specify the format of the response (e.g., "Provide a bullet-point list.")              |
| **A**ction/Goal         | ‚Üí   | Define the action you want the AI to take (e.g., "Generate a 6-month marketing plan.") |
| **C**onstraints         | ‚Üí   | Set any limitations or requirements (e.g., "Focus on digital channels.")               |
| **T**one/Style          | ‚Üí   | Indicate the desired tone or style (e.g., "Use a professional and engaging tone.")     |

**‚ùå Bad Prompt**: Too simple. DOESN'T include elements above.

```
What is Generative AI?
```

**‚úÖ Good Prompt**: Includes SOME or ALL of the elements above.

# Writing Effective Image Generation Prompts

To write effective prompts for images generation, consider the following elements:

- Subject
- Background/Scene
- Adjective
- Style
- Lighting

## Examples

**‚ùå Bad Prompt**: DOESN'T specify any of the elements above.

```
Generate a picture of a cat
```

![](./images/cat-bad-prompt.jpg)

**‚úÖ Good Prompt**: specifies SOME or ALL of the elements above.
| Element | | Example
|----------|----|---------------|
| **Subject** <br>**Background**<br>**Adjective**<br>**Style**<br>**Lighting** | ‚Üí<br>‚Üí<br>‚Üí<br>‚Üí<br>‚Üí| Generate a picture of a cat <br> sitting on a sofa. Behind is a window with curtain. <br>The cat is white. <br>Make the style realistic. <br>Use dramatic lighting.|

# Photo / Video / Icon Libraries

Images/Video generation tools are slow and expensive. Sometimes, it is more efficient to use free stock photo/video/icon libraries.
